KODEX script reference

Patterns, instruments, visuals, and events you can use in the live runtime. Open index.html to perform.

Tempo

tempo(120) sets BPM between 40 and 240. Call once near the top of your script.

Pattern strings

Drums use step patterns: x = hit, . = rest. Note lanes use note names and rhythm mini-language from the parser (braces for sequences, * repeats, ? probability, and more).

kick("x... x.x. ..x. x...")
bass("{c2 eb2 g2 bb1}*2")

Drum lanes

  • kick(pattern), snare(pattern), hat(pattern), clap(pattern)
  • drums(pattern) — kicks by default; configure via chain options.

Melodic instruments

  • bass(pattern) — low monophonic line.
  • lead(pattern) — lead melody.
  • pad(pattern) — sustained harmony.
  • pluck(pattern) — short articulated notes.

Chaining (all instruments)

Each call returns a chainable handle. Common methods:

  • .wave("sine"|"square"|"saw"|"triangle")
  • .gain(n), .attack(n), .release(n), .duration(n), .detune(cents)
  • .delay(time) — also pushes the master delay.
  • .chord(name) — pads: chord quality.
  • .every(steps, fn) — periodic option mutate (advanced).
lead("{c4 eb4 g4 bb4}*2")
  .wave("sawtooth")
  .delay(0.24)
  .gain(0.1)

Visuals

visual("rings")
  .react("kick")
  .color("cyan")
  .trails(0.92)
  .energy(0.2)
  .background("ocean")

Modes include rings, grid, and bars. .react() binds a drum or custom event name so the canvas pulses when that voice fires.

Master FX

fx("delay", 0.22)
fx("feedback", 0.35)
fx("wet", 0.18)
fx("filter", 2400)
fx("master", 0.95)

Events & input

on("space", () => burst("circle", 24))
on("key:a", () => drums("x.x.x.x."))
on("mouse", p => synth("spark").play(p.x))

Keyboard keys emit key:<name>. Mouse moves and clicks feed mouse with a normalized point p (x/y between 0 and 1).

Timing helpers

  • once(beats, callback) — run once after N whole-note beats.
  • repeat(beats, callback) — reschedule every N beats.
  • wait(beats) — returns a Promise (use inside async logic).
  • play(note, { instrument, gain, … }) — one-shot note.
  • chord(root, { chord, gain }) — pad-style chord stab.
  • burst(shape, count) — visual burst on the canvas.
  • stop() — clears loops (runtime also clears on each run).

Synth shortcut

synth("spark").play(0.5, { gain: 0.08 })
synth("drift").play("d5", { gain: 0.04, tone: 2200 })
